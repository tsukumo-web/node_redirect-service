# Node - Redirect Service

## About

Accepts a nameserver to port mapping and applies the port to
the base url.

## Version Support

- 1.0.0 - Flat mapping, ie git.tsu.io -> tsu.io:8888
- 2.0.0 - Nested mapping (not sure what this means yet but I like the idea)
